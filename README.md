# Mark's Personal AI Assistant Workspace

This is the workspace for OpenClaw AI assistant. It contains configuration files, personal notes, and daily logs.

## 📁 Folder Structure

| Folder/File | Description |
|-------------|-------------|
| `diary/` | Daily journal entries (YYYY-MM-DD.md format) |
| `.openclaw/` | OpenClaw internal configuration |
| `.pi/` | Raspberry Pi related files |

## 📄 Core Files

### Identity & Configuration
- **AGENTS.md** - Workspace behavior rules and guidelines
- **SOUL.md** - AI assistant persona and values
- **USER.md** - Personal info about the user (fitness plan, preferences)
- **IDENTITY.md** - AI assistant name and identity details
- **TOOLS.md** - Local tool configuration (cameras, SSH, TTS)

### Bootstrap & Setup
- **BOOTSTRAP.md** - Initial setup instructions (can be deleted after setup)

### Automation
- **HEARTBEAT.md** - Periodic check tasks (workout reminders)
- **backup.sh** - Git backup automation script

### Logs
- **backup.log** - Backup execution log
- **cron-test.log** - Cron job test log
- **last-workout-ask.txt** - Last date workout was asked
- **workout-trigger.txt** - Workout reminder trigger file

## 🔄 Git Sync

## 📝 Daily Notes

Daily journal entries are stored in `diary/` with format `YYYY-MM-DD.md`:
- `2026-02-25.md` - Feb 25, 2026
- `2026-02-27.md` - Feb 27, 2026
- etc.
